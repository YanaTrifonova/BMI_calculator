"User friendly" console BMI calculator.

## Get

Navigate through your terminal where you want to download my BMI Calculator and enter following command

```
$ git clone git@github.com:YanaTrifonova/BMI_calculator.git
```
 
## Run

If you want to run BMI Calculator just run this command in your terminal

```
$ node bmi_calculator.js
```
